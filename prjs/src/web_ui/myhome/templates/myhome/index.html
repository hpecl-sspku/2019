简介

"智能空间门户(英文全称为Smart Space Portal)，是由北京大学软件与微电子学院（SSPKU）高性能计算嵌入式实验室（HPECL）柳翔教授提出，自2008年开始由HPECL研发，期间曾得到诺基亚研究院（中国）的资助

part one.

" 智能空间门户通过分析进入空间内用户本身、所处环境以及周边人群的情景数据（如计划安排、行为习惯和心理状态，地理位置、环境状况、时间、周边物品状况等），感知用户意图，从而自发地为其提供所需要的信息和服务。它将提供一种全新的用户体验，使空间内的用户自发地享受所需要的服务。

part two.

" 智能空间门户是一个统一的、免维护的、嵌入式以及可扩展的服务软件，通过泛在网络管理由嵌入在用户所处场所（居室、办公、休闲及旅行等）的各种计算资源所组成的自治系统：智能空间。

part 3

Latest Works

智能空间门户负责空间内的服务管理、情景数据收集和共享计算资源管理。它将为居室、办公及旅行等智能空间中的用户提供一种全新的用户体验，使其随时随地享受所需要的贴心服务.

项目目前已完成泛在家庭服务演示系统、智能会议室服务演示系统、智能空间门户原型系统以及情景感知引擎原型系统等.

SmartHome系统是在智能空间门户系统的基础上进行进一步的语音交互开发。通过增加语音交互模块，提高用户的体验性.
